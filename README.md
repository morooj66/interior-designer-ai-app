
 AI Interior Designer – Multi-Agent Interior Design Assistant


A full AI-powered interior design assistant that analyzes spaces, understands user needs, and generates complete design plans with images and moodboards.




What it does


This application helps the user get a full professional interior design plan by providing:

 Overview & Summary of the room

 Architect/Layout Plan (space optimization + measurements)

 Furniture Plan (pieces, placement, budget-fit options)

 Color Palette & Materials

 AI Moodboard Generation (furniture + colors only)

 Full 3D Render generated via gpt-image-1

 (Optional) Upload a reference room image

 Built with multi-agent design logic (architect + stylist + color expert)




AI Agents inside the project
Architect Agent → layout, circulation, zoning

Furniture Agent → furniture selection, sourcing, matching budget

Color/Material Agent → color palette + materials

Image Agent → generated moodboard + 3D render
